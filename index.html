ToBoolean / Boolean(something)

  • Any object ‚Äì always true
  • Everything else ‚Äì also true, except for 7 values
    • false
    • null
    • undefined
    • ''
    • 0, -0
    • NaN

ToString / String(something)

  • For primitives ‚Äì add quotes
  • For references ‚Äì call ToPrimitive

ToNumber / Number(something)

  • For strings ‚Äì try to convert to number
  • For booleans ‚Äì true/false to 1/0
  • For references ‚Äì call ToPrimitive

String To Number

  • Empty string ‚Äì 0
  • Remove spaces and try to convert
  • At least one not a number ‚Äì NaN
        Number(' 14 ') // 42
        Number('14px') // NaN
        

           
          pastInt('14px') // 14
          pastFloat('14px') // 14
